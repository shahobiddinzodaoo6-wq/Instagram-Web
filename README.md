**Instagram Web** is a high-performance, feature-rich social media clone designed to replicate the premium experience of Instagram. Built with a focus on real-time interactions, pixel-perfect UI, and modern web standards.

---

## ✨ Key Features

### 📱 Core Social Experience
* **Dynamic Feed:** Real-time fetching and rendering of posts with high-quality media support.
* **Interactive Posts:** Advanced like system, comment management, and post saving functionality.
* **Stories System:** Full-featured stories viewer and creator with image/video support and progress bars.

### 💬 Communication & Networking
* **Direct Messaging:** Real-time chat interface for seamless professional and personal communication.
* **User Profiles:** Comprehensive profile management including post grids, follower tracking, and identity verification.
* **Discovery:** Advanced search and suggestion system to connect with new users.

### 🛠️ Advanced Functionality
* **Media Handling:** Optimized image and video rendering with custom full-screen overlays.
* **State Management:** Highly efficient data synchronization using TanStack Query and Zustand.
* **Localization Support:** Optimized for global use with customizable language and UI settings.

---

## 🛠 Tech Stack

| Component | Technology |
| :--- | :--- |
| **Framework** | Next.js 16 (App Router) |
| **Language** | TypeScript |
| **Library** | React 19 |
| **State Management** | TanStack Query & Zustand |
| **UI/UX** | Tailwind CSS, Ant Design (antd), Lucide Icons |
| **Data Fetching** | Axios with interceptors |
| **Performance** | Next.js Turbopack & ISR |

---

## 🎨 Design Philosophy
The project follows **Instagram's native design language**, prioritizing clean layouts, subtle micro-animations, and a responsive architecture that works perfectly across mobile, tablet, and desktop devices.

## 🚀 Getting Started

1.  **Clone the repo:**
    ```bash
    git clone https://github.com/shahobiddinzodaoo6-wq/Instagram-Web.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run development server:**
